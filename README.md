# Personal-Firewall-Documentation

## 📌 Overview
A lightweight personal firewall that monitors and filters network traffic using Python and Scapy.

## 🚀 Features
- Packet sniffing
- Rule-based blocking (IP, Port, Protocol)
- Logging system
- Real-time monitoring

## 🛠️ Technologies
- Python
- Scapy
- iptables (optional)
